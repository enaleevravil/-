1. Вывести список сотрудников, получающих максимальную заработную плату в своем отделе

SELECT * FROM   employee tab1
WHERE  tab1.salary = (SELECT MAX(salary) FROM employee tab2
		      WHERE  tab2.department_id = tab1.department_id);

--------------------------------------------------
2. Вывести список ID отделов, количество сотрудников в которых не превышает 3 человек

Делаем группировку сотрудников по отделам. 
В условие по полю NAME подсчитываем количество отрудников в сгруппированных данных и вы водим отделы, в в которых полученное число меньше не больше трёх.

SELECT DEPARTMENT_ID FROM employee 
GROUP BY DEPARTMENT_ID
HAVING COUNT(name) <= 3;

--------------------------------------------------
3. Найти список ID отделов с максимальной суммарной зарплатой сотрудников

Выводим 3 отдела с максимальной суммарной зарплатой. Сортировка от максимальной суммы к минимальной

SELECT DEPARTMENT_ID, sum(SALARY) as 'sum_salary' FROM employee 
GROUP BY DEPARTMENT_ID
ORDER BY sum_salary DESC LIMIT 3;
--------------------------------------------------
